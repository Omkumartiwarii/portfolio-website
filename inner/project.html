🏅 College Sports Website


A fully responsive and visually engaging sports event website developed to simplify event discovery and registration for a college-level sports meet. The site is built using HTML, CSS, and JavaScript with animated UI features and a clean, modern layout.

🏠 Modern Homepage UI: Features a branded landing page with navigation, search bar, and animated particle.js background.

📅 Events Listing: All sports events are displayed in a card-based format, each with details and a Join Now button.

📝 Join Functionality: Users can register by filling a form that includes their name, email, college ID, and sport selection. Form validation is handled using JavaScript.

✨ Particle.js Integration: Adds interactive animated particles in the background to enhance user engagement.

📱 Fully Responsive Design: Optimized for mobile, tablet, and desktop using Flexbox and media queries.

🎨 Clean UI: Uses a dark theme and neon blue highlights (#00e5ff) with hover effects and transitions for a modern experience.

Tech Stack: HTML5, CSS3, JavaScript, Particle.js

What I Learned: Improved frontend development skills, DOM handling, UI/UX design, and third-party library integration. Gained confidence in building real-world, interactive, responsive websites.

🚀 A great showcase of frontend and UI/UX skills for academic and project use
